# 第二届Unity中国开发挑战赛官网

> 团结有AI，游我精彩

## 项目简介

这是第二届Unity中国开发挑战赛的官方赛事网站，采用深色科技感设计，融合AI元素与游戏美学，打造沉浸式的赛事展示平台。

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计，包含玻璃拟态、渐变、动画效果
- **JavaScript** - 交互逻辑
- **Three.js** - 3D粒子效果
- **AOS** - 滚动动画库

### 🎨 设计特点
- 深色科技感主调，蓝/紫色系点缀

## 网站模块

1. **Hero首屏** - 沉浸式视觉体验，3D立方体与神经网络动画
2. **赛事概览** - 简介、时间轴、统计数据

4. **奖项设置** - 按赛道区分的奖项金字塔展示

6. **评审阵容** - 专家评委展示
7. **合作生态** - 各级别合作伙伴Logo墙
8. **作品展示** - 往届优秀作品轮播
9. **资源支持** - AI工具/产业资源/孵化资源
10. **FAQ常见问题** - 可搜索的折叠式FAQ
11. **新闻动态** - 赛事新闻、媒体报道、活动回顾

## 快速开始

### 1. 克隆或下载项目

```bash
cd unity-challenge-2026
```

### 2. 直接打开

直接在浏览器中打开 `index.html` 文件即可查看网站。

### 3. 使用本地服务器（推荐）

使用任何本地服务器运行项目，例如：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (http-server)
npx http-server

# 使用 VS Code Live Server 插件
# 右键点击 index.html -> Open with Live Server
```

然后在浏览器中访问 `http://localhost:8000`

## 自定义配置

### 修改颜色主题

在 `css/style.css` 中的 `:root` 部分修改颜色变量：

```css
:root {
    --primary-blue: #0066ff;    /* 主蓝色 */
    --primary-purple: #9933ff;  /* 主紫色 */
    --accent-cyan: #00ffff;     /* 强调青色 */
    --accent-pink: #ff00ff;     /* 强调粉色 */
    /* ... 更多颜色变量 */
}
```

### 修改文字内容

直接在 `index.html` 中修改对应的文字内容。

### 添加/修改评委信息

在 `index.html` 的 `<section class="judges">` 部分添加或修改评委卡片。

### 添加/修改作品信息

在 `index.html` 的 `<section class="showcase">` 部分添加或修改作品卡片。
