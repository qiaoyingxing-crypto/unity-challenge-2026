# 2026 Unity中国开发挑战赛官网

> 团结游AI，智引未来

## 项目简介

这是2026 Unity中国开发挑战赛的官方赛事网站，采用深色科技感设计，融合AI元素与游戏美学，打造沉浸式的赛事展示平台。

### 赛事背景

本届大赛在中国音数协游戏工委的指导下，由Unity中国与上海市徐汇区文化和旅游局联合主办。作为Unity中国官方年度赛事，大赛以"团结游AI，智引未来"为主题，为广大游戏开发者提供超百万奖池与丰富生态资源，旨在发掘和培养中国游戏创作新生力量。

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计，包含玻璃拟态、渐变、动画效果
- **JavaScript** - 交互逻辑
- **AOS** - 滚动动画库
- **Canvas API** - 粒子背景效果

## 网站模块

1. **Hero首屏** - 沉浸式视觉体验，3D立方体与神经网络动画，展示赛事核心数据
2. **赛事概览** - 赛事简介、四大亮点展示
3. **赛道设置** - PC游戏、移动端、小游戏三大赛道详细介绍
4. **奖项设置** - 按赛道区分的奖项金字塔展示
5. **赛事日程** - 完整时间线展示，从报名到颁奖
6. **评审阵容** - 专家评委展示，包含头像、简介
7. **资源支持** - AI工具、产业资源、孵化资源
8. **合作生态** - 各级别合作伙伴展示（指导单位、主办方、支持单位、赞助企业、生态合作伙伴、合作媒体与社区）
9. **作品展示** - 往届优秀作品展示，使用实际游戏截图
10. **FAQ常见问题** - 可搜索的折叠式FAQ
11. **新闻动态** - 赛事新闻、媒体报道、活动回顾
12. **页脚** - 完整的联系信息、社交媒体链接

## 快速开始

### 1. 克隆或下载项目

```bash
cd unity-challenge-2026
```

### 2. 直接打开

直接在浏览器中打开 `index.html` 文件即可查看网站。

### 3. 使用本地服务器（推荐）

使用任何本地服务器运行项目，例如：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (http-server)
npx http-server

# 使用 VS Code Live Server 插件
# 右键点击 index.html -> Open with Live Server
```

然后在浏览器中访问 `http://localhost:8000`

## 自定义配置

### 修改颜色主题

在 `css/style.css` 中的 `:root` 部分修改颜色变量：

```css
:root {
    --primary-blue: #0066ff;    /* 主蓝色 */
    --primary-purple: #9933ff;  /* 主紫色 */
    --accent-cyan: #00ffff;     /* 强调青色 */
    --accent-pink: #ff00ff;     /* 强调粉色 */
    /* ... 更多颜色变量 */
}
```

### 修改文字内容

直接在 `index.html` 中修改对应的文字内容。

### 添加/修改评委信息

在 `index.html` 的 `<section class="judges">` 部分添加或修改评委卡片。

### 添加/修改作品信息

在 `index.html` 的 `<section class="showcase">` 部分添加或修改作品卡片。
确保作品图片放置在 `assets/images/showcase/` 目录下。

### 添加/修改合作伙伴信息

在 `index.html` 的 `<section class="partners">` 部分添加或修改合作伙伴Logo。
合作伙伴图片放置在 `assets/images/` 目录下。
